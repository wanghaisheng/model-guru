About the Role

We're looking for a skilled Next.js Full Stack Developer to join our growing team. You'll be responsible for developing and maintaining our web platform, working on both frontend and backend components, and helping to shape the future of AI model discovery and analytics.

Key Responsibilities

  • Develop and maintain our Next.js-based web application
  • Build and optimize frontend components using React and TypeScript
  • Implement backend APIs and services
  • Work with our data layer (IndexedDB, SQLite) and sync mechanisms
  • Collaborate with the team to design and implement new features
  • Write clean, maintainable, and well-documented code
  • Participate in code reviews and contribute to technical discussions

Requirements

  • 3+ years of experience with Next.js and React
  • Strong proficiency in TypeScript
  • Experience with modern frontend technologies (TailwindCSS, SWR, etc.)
  • Knowledge of backend development and API design
  • Experience with database technologies (IndexedDB, SQLite)
  • Understanding of web performance optimization
  • Excellent problem-solving skills
  • Strong communication skills in English

Nice to Have

  • Experience with AI/ML technologies
  • Knowledge of mobile app development (Capacitor)
  • Experience with testing frameworks (Jest, React Testing Library)
  • Contributions to open-source projects

What We Offer

  • Fully remote work environment
  • Competitive salary and benefits
  • Flexible working hours
  • Professional development opportunities
  • Collaborative and innovative team culture
  • Opportunity to work with cutting-edge AI technologies

How to Apply

To apply for this position, please send your resume and a brief cover letter to support@modelguru.deepseek.diy. Include links to your GitHub profile and any relevant projects or contributions.
