Frequently Asked Questions

Find answers to common questions about Model Guru

General Questions

How does Model Guru collect model data?

+

We collect data through publicly available APIs and information provided by platforms. We update our data regularly to ensure accuracy and timeliness. For more details, check our methodology page.

How often is the data updated?

+

We update our data daily to ensure you have access to the most current information about AI models across all platforms.

Can I suggest a platform to be added?

+

Absolutely! We're always looking to expand our coverage. Please contact us with your suggestions.

How do I switch between dark and light mode?

+

Use the theme switcher button (usually a moon or sun icon) located in the top-right corner of the page to toggle between dark and light themes. Your preference is saved locally.

Features & Usage

How does the model comparison tool work?

+

You can select models from the "All Models" page or the comparison tool section on the homepage. Add models using the 'Add to Comparison' button. Compare selected models side-by-side based on metrics like performance, cost (if available), and features. Premium users can compare more models and access deeper insights.

What kind of reports does Model Guru offer?

+

We offer Daily Insights, Weekly Analysis, and Monthly Strategy Reports. Daily reports track short-term trends, weekly reports analyze medium-term growth and opportunities, and monthly reports provide comprehensive market analysis for long-term planning. Access to full reports is a Premium feature.

Where can I find documentation or tutorials?

+

Check our Blog section for tutorials, guides, and AI model insights. API documentation and detailed user guides will be available soon.

Premium & API

What are the Premium features and how do I get them?

+

Premium features include advanced analytics (detailed metrics, trends), custom report generation, API access, and priority support. You can upgrade to Premium via the "Upgrade" button or links found on comparison/report previews.

Does Model Guru provide API access?

+

Currently, we don't offer API access, but it's on our roadmap. Premium subscribers will be the first to get access when it launches. If you're particularly interested in this feature, please contact us.
